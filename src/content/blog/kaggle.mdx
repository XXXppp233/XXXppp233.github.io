Kaggle 是一个数据科学竞赛平台，提供了丰富的数据集和工具，帮助数据科学家和机器学习工程师提升技能和解决实际问题。如果你有能力验证一个手机号码的话，Kaggle 就能为你的代码提供至多两个 Nvidia T4 GPU 每周 30 小时的免费额度，Kaggle 上使用 Jupyter notebook 来编写代码，可以参照 Colab，在代码块的行首添加 `!`即可执行 Shell 命令。
### 配置
简单查阅信息后得知，这台服务器位于 GCP 的 `us-central1` 区域，也就是 `Iowa`，以 GCP 的抢占式实例定价标准来算，每小时成本约 1.60 美元。8 TB 的硬盘可用空间为 1.1TB 对于小型的机器学习来说绰绰有余了。
|指标|规格|
|---|---|
|CPU|4 vCPU|
|内存|31 GiB|
|磁盘|8 TiB|
|GPU|2 x Nvidia T4|
